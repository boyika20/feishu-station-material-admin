# 分输站物资管理（飞书/海能多维表格 · 边栏插件）

**GitHub：** https://github.com/boyika20/feishu-station-material-admin

面向多站点库存台账的 **PC 边栏插件**：跨站查货、调拨申请、**批量领用过账**（写出库流水后再扣现存量）。

> 单行日常领用请用多维表格原生「按钮」字段 + 自动化；本插件对齐「表内勾选 + 助手批量操作」模式（类似物资验收插件）。

## 功能

1. **总览**：站点 / 库存行数 / 账面合计 / 进行中调拨  
2. **库存检索**：按站、名称、规格、编码、货位过滤  
3. **批量领用**：在「站库存」勾选多行 → 插件「读取勾选」→ 二次确认 → 写出库流水并扣库存（含并发账面校验）  
4. **跨站查货**：按名称+规格看各站可用量，一键填入调拨  
5. **调拨申请 / 审计 / 权限**：申请草稿与只读预览  

## 本地开发

```bash
npm install
npm run dev    # http://localhost:5176
npm run build  # 产物 dist/，base 为相对路径 ./
```

在多维表格 → 插件 → 自定义插件中，将服务地址指向本地或静态托管的 `dist/`（URL 末尾保留 `/`）。

## 表结构约定

插件按表名查找（需与 Base 一致）：

- 站点、站库存、入库流水、出库流水、调拨申请、人员站点权限、修改记录  

字段约定与操作说明见 [`docs/OPS.md`](docs/OPS.md)。
