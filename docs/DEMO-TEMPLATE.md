## 你如何得到「互联网可访问」链接（必须你在飞书网页操作）

1. 打开 https://www.feishu.cn/base → **新建多维表格**（不要用海能正式库）  
2. 按上表建表、贴字段、粘贴演示行（共约 2+8+1+1 = 12 行量级，符合 5–10 行「能演示」要求：库存 8 行即可）  
3. 右上角 **分享** → 开启 **互联网获得链接的人可阅读**（或组织外可读）  
4. 复制链接填上架表单第 8 项  
5. 在该 Base 添加边栏插件，服务地址指向你托管的 `dist/`  

体验路径：站库存勾选 DEMO01 手套+胶带 → 插件「批量领用」→ 过账。
